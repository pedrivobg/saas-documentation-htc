WhatsApp

Envie e receba mensagens de WhatsApp diretamente do CRM.

📱 WhatsApp no [App da Agência]

O WhatsApp é o canal de comunicação mais usado no Brasil. Na plataforma, você pode enviar e receber mensagens de WhatsApp diretamente do CRM, automatizar respostas e manter todo o histórico organizado.

NOTA A plataforma suporta dois tipos de integração. A diferença é importante para escolher o que funciona melhor para o seu volume de mensagens.

ATENÇÃO Se usar API não oficial: não envie mensagens em massa para listas frias. O WhatsApp pode suspender o número por spam.

⏰ Regra das 24 Horas

IMPORTANTE O WhatsApp só permite enviar mensagens livres dentro das 24 horas após o último contato do cliente. Após esse prazo, é necessário usar templates aprovados (API Oficial) ou aguardar o cliente iniciar a conversa.
0h — cliente manda mensagem
→
0h-24h — responda com qualquer texto
→
+24h — apenas templates funcionam

✅ Boas Práticas

  • Envie mensagens apenas das 8h às 20h
  • Use o nome do contato para personalizar
  • Mantenha as mensagens curtas e diretas
  • Respeite o opt-out — se alguém pedir para parar, pare
  • Teste os fluxos com seu próprio número antes de ativar
  • Nunca envie a mesma mensagem para mais de 100 contatos por vez sem warmup

🔗 Conectando o WhatsApp

Para conectar a API oficial, acesse Configurações › WhatsApp (não em Integrações). Você tem duas opções de conexão:

1. Com Coexistência (Recomendado)

Permite usar o WhatsApp tanto no sistema quanto no aplicativo mobile do seu celular.

1

No botão de coexistência, selecione sua BM e um número de WhatsApp Business já existente.

2

Insira o número. O sistema enviará um código para o seu WhatsApp no celular.

3

Clicando no código recebido, a câmera será aberta.

4

Escaneie o QR code exibido na tela do sistema.

5

Pronto! A coexistência está ativada (sistema + mobile).

2. Forma Antiga (Sem Coexistência)

Funciona apenas dentro do sistema, sem acesso pelo app mobile. Exige pagamento prévio (mínimo de ~$10 na Meta).

1

Ao seguir o fluxo antigo, você receberá um SMS de 6 dígitos (em vez de usar a câmera).

2

Insira o código recebido no sistema.

3

O número ficará preso ao sistema e desconectará do seu celular.
