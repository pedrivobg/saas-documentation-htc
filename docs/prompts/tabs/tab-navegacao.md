## Instruções Gerais desta Página

Construir a página seguindo o template padrão definido no Master Prompt:
1. Breadcrumb de navegação
2. Ícone `compass` + H1 como título
3. Parágrafo de resumo
4. Seções H2 com âncoras para o TOC
5. Rodapé com navegação ← Anterior | Próxima →

Todas as imagens devem ser placeholders no formato `[IMAGEM: descrição detalhada do que deve ser capturado]`.

### 📋 Seções do Menu {#menu-sections}

H2 "Seções do Menu".

Tabela 2 colunas — Seção / Para que serve:
| Dashboard | Visão geral dos KPIs e métricas em tempo real |
| Conversas | Inbox de todas as mensagens: WhatsApp, SMS, e-mail, Instagram |
| Contatos | Banco de dados de leads e clientes |
| Calendários | Agendamentos e disponibilidade da equipe |
| Oportunidades | Funil de vendas visual (pipeline) |
| E-mail Marketing | Campanhas e automações de e-mail em massa |
| Automações | Criador de workflows automáticos |
| Sites/Funis | Construtor de páginas e funis de conversão |
| Configurações | Dados da conta, usuários, integrações |

---

### 📖 Glossário Rápido {#glossary}

H2 "Glossário Rápido".

Cards de glossário em grid 2×N, cada um com termo e definição de 1 linha:

CONTATO — qualquer pessoa no banco de dados
LEAD — contato que ainda não comprou
CLIENTE — contato que já fechou negócio
OPORTUNIDADE — negócio em andamento
PIPELINE — trilha visual do funil de vendas
ESTÁGIO — fase dentro do pipeline
TAG — etiqueta para classificar contatos
WORKFLOW — sequência automatizada de ações
TRIGGER — evento que inicia um workflow
SNIPPET — modelo de mensagem reutilizável
CUSTOM FIELD — campo personalizado de dados
ROUND ROBIN — distribuição automática de leads entre consultores
