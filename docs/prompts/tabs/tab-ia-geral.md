### ⚡ IA × Automação Comum {#ai-vs-automation}

H2 "IA × Automação Comum".

Tabela comparativa:
| | Automação comum | IA |
| Resposta | Fixa, pré-escrita | Gerada em tempo real |
| Contexto | Não considera o histórico | Considera a conversa inteira |
| Flexibilidade | Responde exatamente o que foi configurado | Adapta a resposta à situação |
| Quando usar | Mensagens padronizadas (confirmações, lembretes) | Atendimento conversacional, qualificação |

Callout info: "Use automações comuns para mensagens fixas (confirmações, lembretes). Use IA para conversas que precisam de contexto e personalização."
