Tags e Segmentação

Como usar tags para segmentar, filtrar e automatizar com base em grupos de contatos.

🏷 O que são Tags

Tags são etiquetas que você cola em um contato para identificar quem ele é, de onde veio e em que fase está. Pense como as etiquetas de uma pasta física — mas digitais e muito mais poderosas.

novo-lead lead-qualificado reuniao-agendada cliente-ativo no-show upsell-potencial
NOTA A diferença de tag para campo personalizado: tag é uma etiqueta binária (tem ou não tem). Campo personalizado guarda um valor específico (ex: valor da proposta = R$2.000).

Automaticamente

Via workflow (a mais usada)

🔍 Filtrando por Tags

Na lista de Contatos, use o painel de filtros para exibir apenas os contatos com determinadas tags. Você pode combinar múltiplos filtros.

  • Mostrar apenas clientes ativos com tag cliente-ativo
  • Mostrar leads para follow-up com tag followup-pendente
  • Mostrar leads que não respondem com tag lead-nao-responde
