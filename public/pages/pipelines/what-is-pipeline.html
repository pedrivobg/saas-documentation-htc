O que é um Pipeline

O funil de vendas visual e como ele organiza suas oportunidades.

🎯 O que é um Pipeline

Pipeline é o funil de vendas visual. Imagine um quadro com colunas, e cada coluna é uma fase do processo comercial. Cada lead vira um cartão que você arrasta de coluna em coluna conforme avança na negociação.

[IMAGEM: screenshot do kanban de pipeline com cards em diferentes estágios]
NOTA Você pode ter múltiplos pipelines para processos diferentes. Exemplo: um para Prospecção de Clientes, outro para Onboarding, outro para Clientes Ativos.

📊 Estágios do Pipeline

Cada pipeline tem estágios — as colunas do quadro. Você define os nomes e a ordem.

🃏 O Cartão de Oportunidade

[IMAGEM: zoom em um cartão de oportunidade no kanban]
  • Nome do contato
  • Nome da oportunidade
  • Valor estimado do negócio
  • Data de criação / dias no estágio
  • Usuário responsável (avatar)
  • Tags associadas

💡 Por que Usar um Pipeline

Visibilidade

Enxerga onde estão todos os leads de uma vez

Foco

Identifica rapidamente quem precisa de atenção

Previsão

Estima receita futura pelo valor das oportunidades abertas

Automação

Quando um cartão muda de estágio, pode disparar uma automação
