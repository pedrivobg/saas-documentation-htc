📖 Glossário Rápido

CONTATO — qualquer pessoa no banco de dados
LEAD — contato que ainda não comprou
CLIENTE — contato que já fechou negócio
OPORTUNIDADE — negócio em andamento
PIPELINE — trilha visual do funil de vendas
ESTÁGIO — fase dentro do pipeline
TAG — etiqueta para classificar contatos
WORKFLOW — sequência automatizada de ações
TRIGGER — evento que inicia um workflow
SNIPPET — modelo de mensagem reutilizável
CUSTOM FIELD — campo personalizado de dados
ROUND ROBIN — distribuição automática de leads entre consultores
